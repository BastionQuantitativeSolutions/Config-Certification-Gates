# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-07-02

### Added
- Initial release of `bastion-config-certification`.
- `ConfigConflict` and `ValidationResult` dataclasses for structured validation output.
- `validate_config()` example demonstrating single-source-of-truth, threshold, and drift checks.
- `ManualOverrideAudit` class for typed consent and permanent override records.
- Pre-flight certification gate primitives.
- pytest suite and GitHub Actions CI.
